# Personalized Product Recommendation System

## Business Problem

Provide personalized product recommendations to customers based on their historical purchasing behavior to improve customer experience and increase sales.

## Dataset

Olist Brazilian E-Commerce Dataset

## Objective

Recommend products that a customer is most likely to purchase.

## Success Metrics

- Precision@K
- Recall@K
- MAP@K
- NDCG@K

## Deployment Target

AWS EC2

## API Framework

FastAPI

## Experiment Tracking

MLflow
